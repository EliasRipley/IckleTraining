Web Structure Training for Ickle Language Model

HTML ELEMENTS AND STRUCTURE
<h1>Main Title</h1> - This is the most important heading on the page
<h2>Subheading</h2> - This is a secondary heading
<h3>Section Title</h3> - This is a tertiary heading
<p>This is a paragraph of text content.</p> - Paragraph contains main content
<a href="https://example.com">Link Text</a> - This is a hyperlink to another page
<img src="image.jpg" alt="Description"> - This is an image with description
<div class="container">Content wrapper</div> - This is a container for content
<span class="highlight">Important text</span> - This is inline emphasized text
<ul><li>Item 1</li><li>Item 2</li></ul> - This is an unordered list
<ol><li>First</li><li>Second</li></ol> - This is an ordered list
<table><tr><td>Cell</td></tr></table> - This is a table with data

Multiple elements: document.querySelectorAll('.items') - This gets all matching elements

CONTENT ANALYSIS PATTERNS
Main content usually in: <main>, <article>, .content, .post-content
Headlines usually in: <h1>, <h2>, .title, .headline
Navigation usually in: <nav>, .nav, .menu
Sidebar usually in: <aside>, .sidebar
Footer usually in: <footer>, .footer

TEXT PROCESSING FOR WEB CONTENT
Remove HTML tags: text.replace(/<[^>]*>/g, '') - This strips HTML
Decode HTML entities: &amp; becomes &, &lt; becomes <, &gt; becomes >
Normalize whitespace: text.replace(/\s+/g, ' ').trim() - This cleans spacing
Extract URLs: text.match(/https?:\/\/[^\s]+/g) - This finds links
Extract emails: text.match(/[^\s@]+@[^\s@]+\.[^\s@]+/g) - This finds emails
